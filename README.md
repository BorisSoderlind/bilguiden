# Bilguiden

En modern webbapplikation för att jämföra och hitta rätt bil.

## Teknologi

- React 19
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui

## Utveckling

Installera dependencies:
```bash
npm install
```

Starta utvecklingsserver:
```bash
npm run dev
```

Servern körs på `http://localhost:8080/`

## Bygga för produktion

```bash
npm run build
```

## Linting

```bash
npm run lint
```
